# Steve Apps

This project is a website for Steve Apps, showcasing leadership storytelling and coaching services.

## Tech Stack

- **React 19**: Frontend library
- **Vite**: Build tool and dev server
- **Babel**: Compiler infrastructure (for React Compiler)

## Performance Optimizations

### React Compiler

This project uses the **React Compiler** (introduced in React 19) to automatically optimize component rendering performance via automatic memoization.

To see the configuration, check `vite.config.js`.

## Development

1.  Install dependencies:
    ```bash
    npm install
    ```

2.  Start the dev server:
    ```bash
    npm run dev
    ```

## Building for Production

To build the application for production:

```bash
npm run build
```
