# Design Style Guide

## Typography
- **Alignment:** All headings (`h1`, `h2`, `h3`, etc.) should be **left-aligned** by default. Avoid centered text for major section headers unless specifically requested for a specific component (e.g., a centered hero).
- **Font Sizes:**
  - H1: 3.5rem (Desktop), 2.5rem (Mobile)
  - H2: 2.5rem (Desktop), 2.0rem (Mobile)
  - Body: 1.125rem

## Spacing
- Standard section spacing has been adjusted to be tighter than original defaults.
- Top padding for sections with H2 titles is reduced (approx -20px from standard `var(--space-xxl)`).

## Colors
- **Primary:** Dark Charcoal (`#111827`)
- **Accent:** Teal (`#2d8a8e`)
- **Backgrounds:** White (`#ffffff`) or Light Teal (`#f0fdfa`)
